Repeat Statement

  • loops are used to repeat statements. Each repition is called an iteration.
  • While loop repeats statements while a certain condition is true.
  • For Loop gives the abulity to manage what happens just before the loop starts and after each loop iteration has run.
  • variable associated with the loop condition is called a loop counter which is often seen as i
  • While loop will run if the condition is true if not, it will not run and or it will not run at all
  • For loop has three things in the parenthese: Initialization , Condition , final expressions
  • Initialization: only happens once, when the code first kicks off. It's often used to set the initial value of the variable associated to the loop condition
  • condition: is evaluated once before the loop runs each time. if it's true, the code runs. if not, the code doesn't run
  • Final Expression: is evaluated after the loop runs each time. It's often used to update the value of the variable associated with the loop condition, as we saw in the previous example.
