Play with variables

  • to declare a variable you use let and const. declaring a vairiable using let allows the vaule to change where as const doesn't

Variable scope

  • Scope of a variable is the part of the program where the variable is visible and usable
  • variables declared with let or const are blocked scope: their visibility is limited to the block where they are declared.

Expressions

  • An expression is a piece of code that produces a value.
  • An expression is created by combining variables, values and operators.
  • Calculating an expression value is called an evaluation
  • Template Literals are often used to create strings containing the values of some variables. It is also I dentified as ${expression} syntax

Type Conversions

  • expression evaluation can result in type conversions, these are called implicit conversions
  • Javascript is extremely tolerant in terms of typ conversion. However, sometimes conversion isnt possible. If a value failes to convert into a number youll get a NaN (not a number)
  • Sometimes youll convert the value to another type, this is called explicit conversion. Javascript has the Number() and string() commands that convert the value between the parenthesis to a number or a string.
