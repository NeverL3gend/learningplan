Store Data in arrays

  • an array represents a list of elements. An array is an object that has special properties, like "length" to access its size
  • Each value stored in an array associated with a number is called an index
  • you can access a particular element by passing its index within square brackets []
  • you can iterate over an array by using for loop, forEach() method and newer loop for-of
  • Push() method adds an element at the end of the array.
  • unshift() method addss it at the beginning
  • Pop() method and slice() method removes elements from the array

Adding an element to an array

  • push() adds an element at the end of the array
  • unshift() adds an element at the beginning of the array

Removing an element from an array

  • pop() method removes the last array element
  • splice() method has two parameters: first one is the index from which to begin removing, and second one is the number of elements to remove.
