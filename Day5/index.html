Understanding Object Oriented programming

  • OOP is uses objects to contain both data and behavior to create programs
  • "class" keyword it can only contain methods.
  • "contructor" method is called during object creation. it helps initialize the object
  • "this" keyword represents the object on which the method was called on
  • objects are created from a class with the "new" operator
  • OOP is based on prototypes. prototypes are used to share properties and delegate behavior between objects.
  • there are ways to create and link Js objects through prototypes. one is to use the object.create() method.
